# Social Media Ads Campaign Dashboard

## 📌 Project Overview
Developed an interactive Power BI dashboard to analyze social media ad campaign performance using marketing data from platforms like Instagram and Facebook. The dashboard helps evaluate campaign success, engagement, CTR, ROI, and platform-wise performance.

## 📊 Dashboard Highlights
- **Average Impressions:** 61.28K  
- **Click-Through Rate (CTR):** 32.92  
- **Overall ROI:** 301.06K  
- **Total Conversion Rate:** 5.99K  

## 📈 Key Insights
- Instagram generated the highest ROI among platforms.
- Health and Fashion campaigns showed strong engagement performance.
- CTR remained stable throughout the campaign period.
- Facebook and Twitter also contributed significantly to campaign reach.
- Dashboard insights help optimize future ad targeting and budget allocation.
